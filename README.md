## Projets phares

### 1. Infrastructure réseau multi-sites
- **Objectif :** Concevoir et déployer un réseau d’entreprise sécurisé avec segmentation et QoS.  
- **Technologies :** Cisco Packet Tracer, VLAN, OSPF/EIGRP, DHCP, DNS, ACL, DMZ  
- **Points clés :**
  - Routage multi-protocoles et communication inter-VLAN
  - Sécurisation du trafic avec ACL étendues
  - Mise en place de QoS pour prioriser le trafic critique
  - Déploiement de services centralisés (FTP, HTTP) dans une DMZ
- **Compétences acquises :** Planification IP, configuration VLAN, routage, sécurité réseau, test et validation

### 2. Windows Server 2022 – Active Directory PME
- **Objectif :** Centraliser la gestion des utilisateurs et ressources pour 100 postes  
- **Technologies :** Windows Server 2022, AD, DHCP, DNS, GPO, NTFS, partages réseau  
- **Points clés :**
  - Organisation de l’Active Directory avec OU, utilisateurs et groupes
  - Création de partages réseau avec permissions combinées NTFS + partage
  - Déploiement de GPO pour sécuriser et personnaliser les postes clients
  - Redirection automatique des dossiers et limitation du bureau pour certains utilisateurs
- **Compétences acquises :** Administration AD, gestion centralisée des droits, déploiement GPO, test et validation

### 3. Linux – Espace collaboratif multi-groupes
- **Objectif :** Mettre en place un espace collaboratif sécurisé pour deux groupes de travail  
- **Technologies :** Linux (Ubuntu/Debian), gestion utilisateurs et groupes, ACL, bit SGID  
- **Points clés :**
  - Création de dossiers partagés pour chaque groupe avec héritage automatique des droits
  - Gestion fine des permissions via ACL pour chaque utilisateur
  - Tests collaboratifs pour valider les droits d’accès et la sécurité
- **Compétences acquises :** Gestion multi-utilisateurs, sécurité des fichiers, collaboration contrôlée, administration Linux
