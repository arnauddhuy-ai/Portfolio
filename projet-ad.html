Documentation Technique : Infrastructure Windows Server 2022

1. Présentation du Projet

Ce projet consiste à mettre en place une infrastructure centralisée pour une PME de 100 postes. L'objectif est de sécuriser les accès et de faciliter l'administration via Active Directory.

2. Configuration du Serveur (SRV-AD)

  • Système : Windows Server 2022 Core / GUI
  • Adresse IP : 192.168.77.10 /24
  • Rôles installés : AD DS, DNS, DHCP

3. Services Réseau (DNS & DHCP)

Le service DHCP distribue automatiquement les adresses IP (plage .100 à .200). Le DNS assure la résolution de noms pour le domaine entreprise.local.

4. Stratégies de Groupe (GPO) déployées

  • GPO Sécurité : Désactivation du Panneau de configuration pour les utilisateurs.
  • GPO Environnement : Redirection des dossiers "Documents" sur le serveur pour garantir la sauvegarde des données.
  • GPO Bureau : Fond d'écran entreprise imposé aux sessions stagiaires.
