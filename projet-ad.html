Infrastructure Active Directory & Services Réseaux

Déploiement complet Windows Server 2022 pour une PME de 100 postes

1. Présentation du Projet

L'objectif est de centraliser la gestion des utilisateurs et de sécuriser les ressources d'une PME via un contrôleur de domaine Windows Server 2022.

Serveur de référence : SRV-AD
Domaine : entreprise.local
IP Statique : 192.168.77.10/24
DNS : 127.0.0.1 (Bouclage AD)

2. Services Réseaux (DHCP & DNS)

Configuration DHCP

Mise en place d'une étendue pour l'attribution dynamique des IP aux 100 postes clients.

  • Plage : 192.168.77.100 à 192.168.77.200
  • Option 003 (Passerelle) : 192.168.77.1
  • Option 006 (DNS) : 192.168.77.10
ipconfig /renew (Test côté client)

Configuration DNS

Création d'enregistrements hôtes (A) pour la résolution de noms interne.

nslookup SRV-AD.entreprise.local

3. Structure Active Directory

Organisation des Unités d'Organisation (OU) et des groupes pour une gestion granulaire.

Unité d'Organisation (OU) Groupes de Sécurité Utilisateurs (Exemples)
Comptabilité G_Comptabilité cdupont, mbernard
RH G_RH jdurand, sblanc
IT G_IT tpetit, lmartin

4. Stratégies de Groupes (GPO) & Sécurité

Application de politiques restrictives et de personnalisation de l'environnement.

GPO : Comptabilité_Politique

  • Restriction : Interdiction totale d'accès au Panneau de Configuration.
  • Optimisation : Redirection du dossier "Documents" vers le serveur : \\SRV-AD\Compta\%username%

GPO : Stagiaires_Politique

  • Bureau : Masquage des icônes (Réseau, Documents) pour limiter les erreurs.
  • Identité visuelle : Application forcée d'un fond d'écran d'entreprise via partage UNC.
gpresult /r (Vérification de l'application des GPO)

5. Gestion des Partages & NTFS

Mise en place de la méthode AGDLP pour la gestion des droits.

Dossier Permissions de Partage Permissions NTFS
C:\Partages\Compta G_Comptabilité (L/E) G_Comptabilité (Modifier)
C:\Partages\Public Tout le monde (Lecture) G_IT (Modifier) / Tous (Lecture)

Conclusion du TP

Ce projet démontre ma capacité à concevoir et administrer une infrastructure Microsoft complète, de la couche réseau (DHCP/DNS) à la couche applicative et sécuritaire (AD/GPO).
